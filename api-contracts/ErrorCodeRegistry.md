# 业务错误码注册表 (Error Code Registry)

本文档用于统一管理全系统的业务错误码（Business Error Codes），所有错误码均以 `DEP_` 开头，确保全局唯一且不重复。

## 错误码分配原则
- **按模块分段**：每个核心模块分配 100 个错误码空间。
- **语义化**：前两位代表模块，后两位代表具体错误。
- **不可复用**：错误码一旦发布，严禁修改其含义或跨模块复用。

---

## 错误码清单

### 1. 提示词管理 (Prompt Management) [0100 - 0199]
| 错误码 | 含义 | 描述 | 建议 HTTP 状态码 |
|---|---|---|---|
| **DEP_0100** | 提示词不存在 | 找不到指定的 Slug 或版本 | 404 |
| **DEP_0101** | 版本冲突 | 提交的版本号（Version Tag）已存在 | 409 |
| **DEP_0102** | 格式校验失败 | Prompt 内容语法错误或变量不匹配 | 400 |
| **DEP_0103** | 激活操作非法 | 试图激活一个未就绪或已禁用的版本 | 422 |

### 2. 知识库管理 (Knowledge Base) [0200 - 0299]
| 错误码 | 含义 | 描述 | 建议 HTTP 状态码 |
|---|---|---|---|
| **DEP_0200** | 知识库未找到 | 指定的主题或文档不存在 | 404 |
| **DEP_0201** | 上传失败 | 文件格式不支持或解析异常 | 400 |

### 3. 对话与历史 (Chat & History) [0300 - 0399]
| 错误码 | 含义 | 描述 | 建议 HTTP 状态码 |
|---|---|---|---|
| **DEP_0300** | 会话不存在 | Session ID 无效或已过期 | 404 |

### 4. 用户与控制台 (User & Console) [0400 - 0499]
| 错误码 | 含义 | 描述 | 建议 HTTP 状态码 |
|---|---|---|---|
| **DEP_0401** | 身份认证失败 | Token 无效或已过期 | 401 |

### 5. 系统与基础设施 (System & Infra) [0500 - 0599]
| 错误码 | 含义 | 描述 | 建议 HTTP 状态码 |
|---|---|---|---|
| **DEP_0500** | 外部调用异常 | 调用 Python Agent 或模型接口失败 | 502 |
| **DEP_0503** | 服务暂不可用 | 限流或下游服务宕机 | 503 |

### 6. 通用后台任务 (Background Task) [0600 - 0699]
| 错误码 | 含义 | 描述 | 建议 HTTP 状态码 |
|---|---|---|---|
| **DEP_0600** | 任务不存在 | 找不到指定 ID 的后台任务记录 | 404 |
